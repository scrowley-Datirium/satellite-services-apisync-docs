constructor(tasksQueueModel: Model<TasksQueueDocument>, sampleService: SamplesService, serviceTagModel: Model<ServiceTagDocument>)

Task queue service is initialized with available use of sample service This is because tasks in the task_queue relate to a task to be performed by the satellite on a sample The satellite has to know the sample's info, and so this service collects it for it's caller so that the satellite can receive it
